## Pattern Definitions

### 3+ turns on same error
**Severity:** Medium
**Cause:** Not checking learnings, trying same approach
**Action:** Check learnings/, try different approach

### "No, I meant..." repeated
**Severity:** Medium
**Cause:** Specs not detailed enough
**Action:** Improve spec-feature with more questions

### Re-read same file 3x in session
**Severity:** Low
**Cause:** No tracking of what's in context
**Action:** Track reads in session/CURRENT.md

### Context >70% before task 3
**Severity:** High
**Cause:** Too much loaded at session start
**Action:** Review what's being loaded, reduce

### Revision rate >1/task
**Severity:** High
**Cause:** Poor requirements understanding
**Action:** More clarifying questions upfront
