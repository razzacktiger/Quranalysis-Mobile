## How to Update

At /complete-task, add a row:
```
| 01-20 | 4.1.1 | 4 | M | API | 12k | 3k | 15k | 4 | 8 | reviewer | Research task |
```

Fields:
- Date: MM-DD
- Task: Task ID
- Epic: Epic number
- Size: S/M/L/XL
- Type: UI/API/DB/Test/Other
- Input: Input tokens (k)
- Output: Output tokens (k)
- Total: Total tokens (k)
- Turns: Number of turns
- Tools: Tool calls
- Agent: Agent used (or -)
- Notes: Any relevant notes
