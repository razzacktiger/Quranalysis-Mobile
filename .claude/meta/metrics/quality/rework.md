# Rework Analysis

## Rework Definition
Work that needs to be redone due to:
- Bug fixes after commit
- User revision requests
- Failed tests requiring changes
- Architecture changes

## Rework by Cause

| Cause | Count | Avg Cost | Prevention |
|-------|-------|----------|------------|
| Bug found after commit | - | - | code-reviewer |
| User revision request | - | - | Better specs |
| Test failure | - | - | Run tests earlier |
| Architecture change | - | - | code-architect |

## Rework Rate
